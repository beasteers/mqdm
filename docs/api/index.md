# API reference

Start with the small public surface most code uses:

- `mqdm.mqdm(...)`
- `mqdm.pool(...)`
- `mqdm.ipool(...)`

Everything else in this section supports those three entry points.

Use the navigation on the left the same way you would read the package:

- Core APIs for loop and pool progress
- Supporting helpers for common calling patterns
- Output and control for printing, pausing, and logging
- Runtime configuration for the advanced shared-state surface

If you just want the callable surface, the next page is the one to read.
