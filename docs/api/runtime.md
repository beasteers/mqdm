# Runtime and Configuration

Most code never constructs a `Runtime`. This is the advanced surface: isolated
progress state, logging, and runtime-scoped configuration.

::: mqdm.runtime.configure
    options:
      show_root_heading: true
      show_source: false

::: mqdm.runtime.Runtime
    options:
      show_root_heading: true
      show_source: false

::: mqdm.pool.Result
    options:
      show_root_heading: true
      show_source: false

::: mqdm.install_logging
    options:
      show_root_heading: true
      show_source: false

::: mqdm.uninstall_logging
    options:
      show_root_heading: true
      show_source: false
