# Output and Control

These helpers are useful once work is nested, interactive, or producing
terminal output alongside the bars.

::: mqdm.print
    options:
      show_root_heading: true
      show_source: false
      show_symbol_type_heading: true

::: mqdm.pause
    options:
      show_root_heading: true
      show_source: false
      show_symbol_type_heading: true

::: mqdm.sustain
    options:
      show_root_heading: true
      show_source: false
      show_symbol_type_heading: true
