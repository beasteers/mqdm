# Core APIs

These are the main public APIs. If you are using `mqdm` as a `tqdm`
replacement, this covers most use-cases.

::: mqdm.bar.mqdm
    options:
      show_root_heading: true
      show_source: false
      show_symbol_type_heading: true

::: mqdm.pool.pool
    options:
      show_root_heading: true
      show_source: false

::: mqdm.pool.ipool
    options:
      show_root_heading: true
      show_source: false
